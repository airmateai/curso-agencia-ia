# SOP — Onboarding completo de un cliente nuevo

Procedimiento paso a paso desde que el cliente confirma hasta que el proyecto entra en fase de mantenimiento. Sigue este orden siempre, sin saltarte pasos aunque el cliente tenga prisa.

## 1. Confirmación (día 0)
- [ ] Mandar email de confirmación de proyecto (recurso 01) o contrato (recurso 02 si es proyecto grande)
- [ ] Mandar link de pago del 50% del setup (Stripe)
- [ ] No empezar ningún trabajo hasta que el pago esté confirmado

## 2. Sesión de información (día 1-3, tras el pago)
- [ ] Agendar llamada de 30 minutos
- [ ] Preguntar: nombre exacto del negocio, cómo quiere que se llamen las secciones, colores/marca, logo (pedir en alta resolución), correo para notificaciones, número aproximado de clientes/citas al mes
- [ ] Preguntar específicamente por el problema principal (revisar notas de la llamada de descubrimiento)
- [ ] Confirmar fecha estimada de entrega

## 3. Construcción (día 3 en adelante, según plazo acordado)
- [ ] Partir siempre de la plantilla base, nunca desde cero
- [ ] Adaptar: nombre, colores, campos específicos del sector, datos de contacto
- [ ] Probar el flujo completo tú mismo antes de enseñar nada al cliente
- [ ] Escribir el documento técnico interno de 1-2 páginas (decisiones, qué hace cada parte)

## 4. Entrega y formación (el día acordado)
- [ ] Videollamada de 30-45 minutos enseñando el sistema en vivo
- [ ] El cliente prueba delante de ti las 3 funciones que más va a usar
- [ ] Grabar la sesión o un vídeo corto que el cliente pueda consultar después
- [ ] Confirmar por escrito el pago del 50% restante
- [ ] Mandar link de pago de la mensualidad

## 5. Primera semana
- [ ] A los 2-3 días, contactar preguntando si algo no ha quedado claro
- [ ] Resolver cualquier fricción de uso real (no funcionalidades nuevas — eso se presupuesta aparte)

## 6. Primer mes
- [ ] Pedir testimonio/reseña si el cliente está satisfecho
- [ ] Pedir referido
- [ ] Anotar en tu propio sistema: qué incluye su mantenimiento, links de pago usados, y dónde está guardado el documento técnico
- [ ] Agendar el primer check-in mensual
